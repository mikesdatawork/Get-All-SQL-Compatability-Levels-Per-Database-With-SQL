## About-Process

<p>Here's some quick SQL Logic to return all the Compatability Modes per Database.</p>      


## SQL-Logic
```SQL
use master;
set nocount on
 
select
 'server' = serverproperty('servername')
, 'database' = name
, compatibility_level
, 'version' = 
 case compatibility_level
 when 65 then 'SQL 6.5'
 when 70 then 'SQL 7.0'
 when 80 then 'SQL 2000'
 when 90 then 'SQL 2005'
 when 100 then 'SQL 2008/R2'
 when 110 then 'SQL 2012'
 when 120 then 'SQL 2014'
 end
from
 sys.databases
order by
 name asc
```
